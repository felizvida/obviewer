# Code of Conduct

Please keep this project welcoming, respectful, and constructive.

## Expected Behavior

- Assume good intent.
- Give clear, kind, actionable feedback.
- Focus critique on ideas and implementation details, not people.

## Unacceptable Behavior

- Harassment or personal attacks
- Discriminatory language
- Dismissive or hostile participation

The repository owner may moderate discussions and contributions to keep collaboration healthy.
